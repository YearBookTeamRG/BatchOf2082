✏️ QUICK EDITS
---------------

ADD YOUR LINKS (in index.html):
  Search for  YOUR_WHATSAPP_LINK   → replace with your WhatsApp group link
  Search for  YOUR_DRIVE_LINK      → replace with your Google Drive link
  Search for  YOUR_YEARBOOK_LINK   → replace with your Digital Yearbook link

CHANGE THE HOMEPAGE MESSAGE (in index.html):
  Find <p class="message-text"> and rewrite the text.
  Line breaks you type show exactly as written — no special tricks.

ADD TEACHER NAMES (in teachers.html):
  Find each  Teacher Name 1  through  Teacher Name 6  and replace them.

ADD TEACHER MESSAGES (in teachers.html):
  Find each <p class="teacher-message placeholder"> and paste/type the message.
  ✅ Just paste — line breaks between paragraphs are preserved automatically!
  ✅ After adding the real message, remove the word "placeholder" from the class:
     Change:  class="teacher-message placeholder"
     To:      class="teacher-message"
     (This changes the text from faded to bright)

ADD TEACHER ROLES (in teachers.html, team.html):
  Same process — find the placeholder text and replace it.

CHANGE TEAM MEMBER ROLES (in team.html):
  Find  Lead Designer / Content Writer / Coordinator
  and replace with the actual roles.


📸 PHOTOS — ZERO CODE NEEDED
-----------------------------
1. Take your photo files
2. Rename them exactly as listed above (e.g. teacher-1.jpg)
3. Drop them in the images/ folder
4. Refresh the website — the placeholder disappears automatically!


🌐 HOW TO OPEN THE WEBSITE
----------------------------

Works offline! (Fonts need internet on first load.)
